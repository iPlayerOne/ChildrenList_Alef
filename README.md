# ChildrenList

Тестовое iOS-приложение на UIKit для заполнения информации о пользователе и его детях.

## Описание

Приложение содержит один экран с формой:

- поля для ФИО и возраста пользователя;
- список детей с возможностью добавить до 5 записей;
- поля имени и возраста для каждого ребенка;
- удаление отдельной записи ребенка;
- полная очистка введенных данных через подтверждение.

## Стек

- Swift
- UIKit
- Auto Layout
- UserDefaults
- OSLog
- MVVM

## Архитектура

Проект разделен на несколько слоев:

- `Models` - модели данных пользователя и детей;
- `Views` - переиспользуемые UI-компоненты;
- `ViewControllers` - экран формы и обработка UI-событий;
- `ViewModels` - бизнес-логика экрана;
- `Storage` - сохранение и загрузка данных;
- `Utilitites` - вспомогательные классы и расширения.

## Возможности

- Ввод и редактирование данных пользователя.
- Добавление детей по кнопке `+`.
- Ограничение списка детей до 5 записей.
- Автоматическое скрытие кнопки добавления после достижения лимита.
- Удаление любого ребенка из списка.
- Очистка всех данных после подтверждения.
- Сохранение данных между запусками приложения.
